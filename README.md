# Gemini Desktop

Gemini Desktop to projekt, który ma na celu stworzenie przyjaznego dla użytkownika środowiska pracy na komputerze. Naszym celem jest udostępnienie narzędzi, które zwiększą produktywność oraz poprawią komfort użytkowania.

## Funkcje
- Intuicyjny interfejs
- Wysoka wydajność
- Możliwość dostosowania do indywidualnych potrzeb
